    • 类型转换

      • StringToNumber

      • 装箱转换
        • 基本类型ToObject
        • 包装类强制类型转换
          • Object()
        • Object.prototype.toString.call(instance)
          • 比instanceof更加准确
      • 拆箱转换
        • ObjectTo基本类型
        • 先拆箱再转换
          • ObjectToNumber
            • valueOf()
            • toString()
            • 若没有返回基本类型，拆箱失败，TypeError
          • ObjectToString
            • toString()
            • valueOf()
            • 若没有返回基本类型，拆箱失败，TypeError
        • Symbol.toPrimitive
          • 若定义了该属性则覆盖原有行为

    • JavaScript面向对象编程

      • JavaScript对象的4个特征：标识性、有状态、有行为、动态性

        • 对象具有唯一标识性
          • 对象具有唯一标识的内存地址
        • 对象有状态(属性)
        • 对象有行为(属性，js中函数是特殊对象，所以也抽象为属性)
        • 对象能够在运行时为操作对象的状态和行为(js特有)

      • JavaScript对象的两类属性：js中的属性并非只有名称和值，用一组特征来描述属性，特征称为属性描述符

        • 数据属性：接近于其他语言的属性
          • value
            • 值
          • writable
            • 是否可被赋值
          • enumerable
            • 是否可枚举
          • configurable
            • 是否能delete
            • 是否能切换属性类型
        • 访问器属性
          • getter
            • 函数或undefined，取属性值时被调用(object.property)
          • setter
            • 函数或undefined，设置属性值时被调用(object.property = value)
          • enumerable
          • configurable
        • Object.getOwnPropertyDescripter(object, key)
        • Object.defineProperty(object, key, value)

      • JavaScript对象的结构：是一种关于属性的哈希结构，属性名是key，属性描述符集合是value

        • 对象的属性名为key
          • 字符串
          • Symbol
        • 上述两种属性特征值对象为value
          • {value:1, writable: true, enumerable: true, configurable: true}

      • JavaScript中的原型

      • JavaScript中的类

  • 实例

    • JavaScript中的对象分类：可以分为宿主对象和内置对象两种
      • 宿主对象：由JavaScript宿主环境提供的对象
      • 内置对象：由JavaScript语言提供的对象，分为固有对象、原生对象和普通对象
