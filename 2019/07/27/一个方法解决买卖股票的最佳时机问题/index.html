一个方法解决买卖股票的最佳时机问题

发表于 2019-07-27 | 分类于 algorithm | 阅读次数：

买卖彩票最大收益系列问题

121. 买卖股票的最佳时机
只能买卖一次的最大收益

122. 买卖股票的最佳时机 II
买卖无数次的最大收益

123. 买卖股票的最佳时机 III
买卖2次的最大收益

188. 买卖股票的最佳时机 IV
买卖k次的最大收益

309. 最佳买卖股票时机含冷冻期
买卖无数次且卖完隔一天才能购入新的股票

714. 买卖股票的最佳时机含手续费
买卖无数次且每次交易有手续费

题目意思是给出一个整数数组prices，数组下标代表天数，元素代表那一天股票的价格，每天可以做购入和卖出或者什么不干，卖出必须在购入之后，而且不能同时持有两个股票，即必须卖出前一个股票才能买下一个股票

例如：prices = [3,3,5,0,0,5,1,2]

121题的答案就是5，0买入，5卖出
122题的答案就是8，3买入5卖出，0买入5卖出，1买入2卖出
123题的答案就是7，3买入5卖出，0买入5卖出
188题的答案就是上述的通用解法，将k传入函数
309题的答案就是7，3买入5卖出，第二个0买入5卖出
714题的答案要将fee传入函数

思路

通用解法是动态规划

1.首先定义状态整体可以考虑成一个三维DP问题，用dp[i][k][j]表示
    1) i表示天数，范围是0 ~ prices.length-1

    2) j表示当天的持股状态，范围一般是0和1,0代表当天不持股，1代表当天持股。如果是含冷冻期的话可以再加一个状态2表示当天是冷冻期

    3) k表示当前完成交易的次数，范围是0 ~ k，上述题目分别k=1，k=k-1(infinite)，k=2，k=any四种情况

2.定义好了DP状态，就要写状态转移方程，即分析如何变化
    1) 不持股的当天，一共会有三种情况，前一天也没持股或者当天卖掉了股票，还有种可选状态，当天是冷却期。所以不持股的当天的最大收益可以这样表示(dp[i][k][0])：
    无冷却期：dp[i][k][0] = max(dp[i-1][k][0], dp[i-1][k][1] + prices[i] )
    含冷却期：dp[i][k][0] = max(dp[i-1][k][0], dp[i-1][k][2] )

    2) 持股的当天，一共会有两种情况，前一天也持股了或者当天买入一只股票。所以持股的最大收益可以这样表示(dp[i][k][1])：
    dp[i][k][1] = max(dp[i-1][k][1], dp[i-1][k][0] - prices[i])

    3) 第三个可选状态，冷却期当天，一共就一种情况，前一天卖出了一只股票。所以冷却期当天的最大收益可以这样表示(dp[i][k][2]):
    dp[i][k][1] = dp[i-1][k][1] + prices[i]  
3.状态方程也写完了，下面可以来写代码了，我这里用的是JavaScript实现，首先需要一个数据结构来存储这些状态，我们使用数组，比较方便，不过要注意的是JavaScript中需要手写初始化多维数组
