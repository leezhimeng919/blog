栈数据结构

  • 创建栈

    • 声明stack类
    • 用数组来保存栈的元素
      • items = []
    • stack类应该有的方法
      • push(elements)添加一个或几个新元素到栈顶
        • items.push(elements)
      • pop()移除栈顶的元素，并返回被移除的元素
        • items.pop()
      • peek()返回栈顶的元素
        • items[items.length-1]
      • isEmpty()查看栈是否为空，空为true，非空为false
        • items.length === 0
      • clear()移除栈中所有元素
        • items = []
      • size()返回栈里的元素个数
        • items.length

链表

链表的数据结构

  • 链表是一组节点组成的集合。除了最后一个节点每个节点都有一个后继节点
  • 链表的构造函数：Node类和LinkedList类
  • 链表的方法：
    • 增加新元素 ：append(element)
    • 插入新元素 : insert(element, position)
    • 按位删除元素 : removeAt(position)
    • 按值删除元素 : remove(element)
    • 查找元素位置 : indexOf(element)
    • 查看链表长度 : size()
    • 查看head元素 : getHead()
