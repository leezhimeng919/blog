关于有效学习

发表于 2019-06-11 | 分类于 软技能 | 阅读次数：
高效学习的定义
找到最 适合自己的 学习手法，在 短时间内集中 注意力，以 解决难题为 目的(设定目标)着手学习。需要勤加练习，有时候可能需要向 名师求助。
七个关键字
: - 适合自己
: - 学习手法
: - 短时间
: - 注意力
: - 解决难题
: - 设定目标
- 名师
学习方式分两种
: 自然主义：核心是 模仿。注重文本的本身。文本就是一门知识或者技能它表面上看起来的样子。遇到什么学什么，积少成多，以量取胜。学完了就能运用了。
结构主义：核心是 创造。侧重对结构和交互关系的一种认识，它提倡透过表面的现象去寻求底层的关系。能够举一反三。是一种质变。
练习不必顾全大局
有效变高效的两种方法：一是避免无效学习，二是用更好的学习方法。
更好的学习方法：把目标拆碎，针对那些细节进行反复联系。不要一开始就顾全大局。
碎片化学习的真相
碎片化学习一定要基于 搜索，意思就是带着困惑和问题去执行碎片化学习。为的就是把碎片链接起来。首先要有目的的去拾获这些碎片而不是乱接。碎片化学习作为一个成年人的学习的补充手段是有很大力量的。有一种假的碎片化学习叫做收藏，如果收藏了不看的话，它并不是你的。你必须要破除掉这种幻觉。
选择正确的学习方法之后，把这个目标拆碎，针对细节进行反复练习。利用碎片化的方式挤出更多的时间练习。
成年人不需培养兴趣
人做事的驱动来自两个方面：来自内部（兴趣、虚荣心等等），来自外部（完成任务之后的奖励）、所以成年人不需要培养兴趣，应该用任务来驱动，比如职业强迫、以教为学。
职业强迫
心态上要有一种自觉，自己来聘请自己。
以教为学
一方面自己对自己设置的一个任务，一个目标，另一方面重复，梳理，学的效果更好。带着去教别人的心态参加学习，效果是最好的。
拖延症的表现与本质
本质是注意力无法集中。学名叫注意力缺陷多动障碍。
关于拖延症，应该认识两点：第一，任何人都有拖延症，只不过有些事拖延，有些事不拖延。因为一个人总不可能一直什么都不干。你之所以一直拖延，是觉得拖延的事其实也没那么重要。第二，就算是自认为拖延症和被别人认为有拖延症的人也没有被拖死，大家都活着好好的。拖延症没有想象中那么严重。
所以其实你拖延的那些事，拖延它们对你影响根本不大，你不是真正的想去做这些事。你根本没有拖延症，只要你有一些是能够按时完成的事，你就没有拖延症。
将任务“严峻化”
解决拖延症只有解决那些对你来讲真正严峻的任务。你要通过做成一件事来突破，然后形成习惯就可以了。把那些自己不喜欢却必须要做的事给做了。才能让我们成长。
设定小目标，形成习惯
健身时不能总想着要有一个健硕的体魄。肌肉男的养成一定伴随着一定程度的痛苦，这些人的长期目标非常明确，别人看起来会觉得他过程很苦，他们自己不觉得苦，他们就能坚持下来。长期目标如果它太长了，太宏达了，会增加这个过程当中的痛苦。
当学习很痛苦的时候，去学一些找到感觉就能掌握的技能。先去学一些技巧性比较强的东西来增强自己的信心。
成年人读书一定要学会速读，不能再精度。带着问题去看书，这样当解决问题时会增强自己的信心。
在衣食住行上训练专注力
平常的生活本身不能训练人的注意力，但是在平常的生活中去营造仪式感，这样会提升我们的注意力。
衣->购物
食->吃饭
住->睡眠
行->灯光
购买昂贵的学习工具
训练专注力的杀手锏，如果学一样东西需要工具的话，就买贵的。稍微让自己心疼一点，就算达到效果了。
因为便宜的东西本身就不是靠专注力制造出来的，用一个东西需要感知到那种专注力，才能对自己的专注力形成一种训练。好处还有一方面就是没余钱去买那些便宜的东西了。也就是说我们拥有的东西少一些，东西越多越不能使人专注。
学会一个人好好吃饭
尽量吃点好的，少吃速餐，少点外卖，少吃自助餐
使用射灯
使用射灯时其他环境都是黑的，有种舞台的感觉，在舞台上会比坐在观众席里更专注一点。
进入高质量的睡眠
睡眠不够人是很难做到专注的。
设定有效目标
找准学习的切入点。
直奔大师
不必从基础开始
要去到哪里（有兴趣），就从哪里开始（任务驱动）
我听完之后很喜欢的一句话：怯场是最糟糕的自觉，一个人他胆子变得比之前大一点，能走的路可能就会长一点。
给自己制造反馈
既然没有这条路让我们走，我们就自己开创一条路来走。必须先有成果。基于成果来发现问题。
当你觉得你学了一些东西之后，一定要想办法讲出来，或者应用出来。
制造文字反馈
写摘要：对照文本写，而不是脱稿写；用自己的语言写，不是摘抄。因为写的时候就是在复习。

制造语音类的反馈（录音）、制造肢体类的反馈（录像）

怎么突破学习的瓶颈
当你遇到瓶颈的时候，首先要鼓励自己，已经走到一定程度了，有些人还没出发。心态问题解决后，想想可能是因为眼界和格局不够所以形成的瓶颈，也有可能是不够专注。
瓶颈是瓶颈，而不是极限。
解决心态问题
自信心方面可以从游戏上尝试突破：技术层面有问题就练技术，心态问题（上分上不上去，总是卡在4000分，可以在4000分时故意输，掉到3900，遮住分数，继续玩。）
扩大涉猎范围
不断积累、多感受、功夫在诗外。
更加专注
把注意力集中在一个更狭窄的范围内，以做到不去分心。
相信一通百通。
